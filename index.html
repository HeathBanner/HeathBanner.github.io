Trivia

The trivia game is a multiple choice question game with a time limit of 30 second per question. At the end, you will be presented with the number of corect, incorrect and amount of times the clock ran out.

API

This application grabs gifs from Giphy. The user has the ability to add gifs to their "favorites". At the top, there's an input form that lets you add new buttons to search for different gifs. The "Movie" button allows the user to grab movies, shows and episodes. It'll Display the title, poster, rating and plot. Charlotte's current weather is Displayed at the top right of the screen. The colors/mood of the website changes according to the weather of the city.

RPS

RPS(Rock Paper Scissors) is a turn-based game that requires two players in seperate browsers. I've implemented the Firebase Real-Time Database to allow multiplayer functionality. The users will have to "login" in order to be able to play the game. The username is saved within local storage and temporarily within the database. A chat box has been designed in to the game to allow the players to communicate with one another. Once a user disconnects, the game is then restarted and awaits for two users to login once again...
